通往鸟烧窝广播村的路上

王一通

老唐，你晓得的，我天天要读报纸。最早的时候 报纸上有个文艺天地，后来那个栏目取消了，没得内容，我就只有读我写的诗。结果镇上有人来表扬了我，说我是丰富了文化的生活，还送了我一副羽毛球拍，但是没有球。傍晚的时候，我把羽毛球拍挂在树上，它们邦邦邦地，反复地碰在一起，我看到是风穿过了它们。

老唐，今天跟着我们那个小屁孩

老唐，今天跟着我们那个小屁娃儿，你要小心，他是个危险人物。以前他拿石头打我，我控制住他，问他原因，他说是因为喜欢听石头敲在锅上那个声音，铛！一声。我给他买了一盒擦炮，就想希望他喜欢上别的声音。结果后来他用擦炮 炸 我。

老唐，你从北京来

老唐，你从北京来，应该认识 小鹿姐姐吧，有一次她在电视里喊我给她写信，我坐在板凳上想了三天，还是对她无话可说。你回北京了，可 不可以帮我带一封空白的信给她，然后说声抱歉，你就说，虽然我见过很多发光的瞬间，但是 心从来没有住进过哪座城堡。

老唐，你睡的床是我爷爷的

老唐，你睡的床是我爷爷的，他老了以后 就经常梦游，越走越远。有天晚上他把我喊醒，说他要出远门了，以后要我自己弄饭吃。我 想点点头，但是我看见他眼睛是闭着的，就没点。说完他就走了，很着急，鞋子都没穿。我就 跟上去，他说：你注意 哦，狗的叫唤 会在山之间 碰撞叠加，荡起的不是土，是古代的孢子。天亮的时候，是离开淤泥的最后时机，要不然身体就会被稻草生根，一直困到太阳沉没。

老唐，其实你们来的还不是时候

老唐，其实你们来的还不是时候，如果说你们夏天来，这里就会很茂盛，水凼凼里面 也会有草长出来，那个时候人和牛，区别就会变得很小。都是翻开新土，然后偿还自然。最好耍的就是 晚上七八点钟的时候，虫的上辈子 从地下爬出来，趴在树干上蜕壳，找不到进化方向。但是 不用多久，它们就都叫唤起来，非常的热闹。这个时候，如果说我想融进去，就会大声地读诗。诗的意思 跟它们叫唤得 一样，说来说去，喊破喉咙，无非都是一句:来了哦，来了哦，我来了 哦。如果 累得停了下来，就会觉得那种喧闹 千奇百怪，震耳欲聋，甚至偶尔还会流出眼睛水，但是也不晓得为啥子，就还是 舍不得先走，一直要等到 山那边黑色的风 把 我们 吹散。
