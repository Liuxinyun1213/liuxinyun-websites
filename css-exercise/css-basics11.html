be present

展览创作者们经历了独特的大学生涯，在大量线上与线下学习模式交替而产生的混沌里，通过思考、表达、创作、实践，构建出个体和设计的轮廓与内核。以思辨与设计介入当下，研究设计语言创新与发展，梳理、解构、并以设计手法展开社会、科技及文化议题，又或提出可行性解决方案。通过毕业设计展示，23位创作者共同呈现了她们的在场与真实。
