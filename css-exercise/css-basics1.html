拳打徐天顺

在氤氲的梦中赶夜路，

啼哭从地里发芽，

在历炼中求生的人 掐住脖子，

捧出记忆 与历史的混血种，

醉汉的眼眶水声洪亮，

背着凿子远走，探寻杯底秘境，

晃荡出忠于自然的铁花和石像，

等到泥潭的伤口结 痴，

鱼和鹿 便每日亲吻，上岸或下山。

灶台上，年兽吞下偷嘴的猫，

日历上记满山野恩仇。

雪地里的脚印 在日落时露出潮湿的弹响，

无人看管的钓竿 被鱼群悉心照料。

蜉蝣用一日道行上岸，远山摧崩。

红脸 村民 在稻田里大声叫喊，以物易物。

先是偷鸡摸狗的恨，

然后用无人察觉的爱，

支付一场烂醉。

屋前的雨季，

少了滚烫的肌理和坚硬的坠落。

水滴是云里被冲散的鱼群，腮借给树的呼吸。

路过的人满心期待，舒展羽毛，

耳畔厮磨，相互捏合形状。

用初遇 代替赴约。

分享王一通的单曲《拳打徐天顺》
