脚踢刘永富

通往河底的阶梯在午夜展开， 携带年轻的照片 入睡， 沿途捡起石底的蝉子， 蹒跚学步，牙牙学语。 溪流灌进发间，冲洗午后 沉默的汗滴， 来不及学会游泳的少年，不愿 归家。 眼泪涂抹的石块 生出星星藓，生产气泡。 宇宙的风 总会在某一瞬间 突然停 住， 柔软的金属，散成雾 的 滩涂。 山林 用奔跑勾出静止的壳， 幼鸟在娘胎里流浪，回忆飞翔。 倦意 与暴雨青黄不接，秋虫如墙倒。 镶在石中的陶瓷 结出露珠， 倒影里，花布为帆，草席顺流而下。 归人拔掉坟前醉去的草， 疼痛响彻树 底的迷宫。 啸叫的哨音攀爬山脊， 鼓动纸灰，向天空索吻。 山谷里住满孤 独的旅客， 他们把甜美的爱 灌进瓶中。 夭折的孩子 吐出萤火虫， 牵引供桌上的亡魂。 小心地避开犬吠， 顺倒雾气流淌。 影子 浮在树的荒海。
